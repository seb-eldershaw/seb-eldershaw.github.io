JavaScript DOM

- The JavaScript DOM -

What is the DOM? DOM stands for Document Object Model, and is a representation of the HTML that makes up a webpage. What makes the DOM special is that it allows the webpage to be modified using JavaScript. Without the DOM, JavaScript has no idea about webpages, elements, or styles. A browser uses JavaScript to make changes to a webpage, and the DOM is the part that shows those changes, which is why the DOM often looks different to the original HTML used to make the webpage.
