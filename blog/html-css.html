Element positioning types in HTML

There are three main ways to position elements in HTML, namely, Relative, Absolute, and Fixed. Note that elements will overlap other elements, in order of definition.

Relative Positioning

An element that uses relative positioning starts at its default position, and can then be moved from there by entering top, bottom, left, and right values. Here, this blue box is moved 20 pixels to the left, and 10 pixels down.

Absolute Positioning

An element that uses absolute positioning is positioned relative to the parent element that it is part of. If the element does not have a parent element, then it defaults to the page element. This red box is positioned 50 pixels right from its normal position in its parent element, the green box.

Fixed Positioning

An element that uses fixed positioning always stays in the same position, relative to the viewport. This means that wherever it is placed on the screen, it will stay there, even if the page is scrolled. Elements that use fixed positioning can overlap other elements, so be aware when using this type of positioning.
